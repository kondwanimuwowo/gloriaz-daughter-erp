## Image Requirements
- **Format**: JPG or WebP
- **Main images**: 800x1200px minimum
- **Detail shots**: 600x800px minimum
- **Lifestyle shots**: 1200x800px minimum
- **Optimize**: Compress to < 200KB per image

## Quick Download Instructions
1. Go to unsplash.com
2. Search using the terms above
3. Download medium or large size
4. Rename to match the naming convention
5. Place in `public/images/products/`
6. Optionally compress using tinypng.com
